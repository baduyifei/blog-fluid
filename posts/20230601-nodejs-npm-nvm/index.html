node.js、npm、nvm的介绍安装与使用

一、nvm

nvm是一款非常方便的Node.js版本管理器，它是 node.js 和 npm 的版本管理器，您可以使用nvm或n来管理多个Node.js版本，并切换到你所需要的版本(8.x-12.x)。 但是它需要手动安装（非自带）。

1.1 安装 nvm

要在电脑上安装nvm，可以直接使用Homebrew安装： brew install nvm

Homebrew会自动将nvm的安装路径添加到PATH中，安装完成后可以直接使用nvm命令。（使用 homebrew 的方式安装，不需要手动将nvm的路径添加到PATH中，这是最方便的一点）

1.2 使用 nvm

使用nvm管理/切换您的Node.js版本的方法：

1）临时切换的方法：

  • 安装完成nvm后，就可以使用nvm管理您的Node.js版本了。

  • 例如:

  • nvm install 8.16.0 # 安装Node.js 8.16.0

  • nvm use 8.16.0 # 切换到8.16.0

  • nvm ls # 列出已安装的Node.js版本

  • 我的使用：

  • 注意：这只是临时切换至该版本，重启终端后还是会恢复至 default 别名指向的版本(如果设置了的话)。

2）永久切换的方法(设为默认版本)：

  • 如果要设置 nvm 的 default 版本为 12.22.12，
    则执行： nvm alias default 12.22.12

    这会将 default 别名指向 12.22.12 版本。

  • 重启终端。

  • 重启后，nvm 会自动加载 default 别名,使用 12.22.12 版本。

三、node.js

node.js 是项目开发需要用到的代码库。

注意：node.js 和 npm 会一起安装。也就是我们在安装 node.js 的时候，npm 也会自动被装上。但是如果需要 nvm，则需要单独安装。
